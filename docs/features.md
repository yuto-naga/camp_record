# 機能概要 (Functional Overview)

本プロジェクト「Camp Record」は、個人のキャンプ体験を記録し、管理するためのWebアプリケーションです。

## 主な機能

### 1. キャンプ記録の管理 (Visited Records)
行ったことのあるキャンプ場の情報を詳細に記録できます。
- **基本情報**: キャンプ場名、訪問日、料金
- **評価システム**: 5段階の星評価 (Star Rating)
- **詳細メモ**: 感想や周辺施設情報をテキストエリアで入力可能
- **写真保存**: キャンプ場の写真をアップロードして視覚的に保存

### 2. 行きたいリスト (Wishlist)
これから行きたいキャンプ場をメモしておくことができます。
- 気になるキャンプ場をリストアップし、実際に訪問した後は「訪問済み」に更新可能

### 3. 画像アップロードと最適化
- スマートフォンやPCから写真をアップロード可能
- **自動圧縮**: `browser-image-compression` を使用し、ブラウザ側で画像を軽量化してから送信することで、ストレージ容量の節約とレスポンス向上を実現

### 4. レスポンシブ対応
- **モバイルファースト**: スマートフォンでの利用を想定したUI設計になっており、キャンプ場での現地入力もスムーズに行えます
- **デスクトップ対応**: PCの広い画面でも快適に一覧を確認可能

### 5. 日本語ローカライズ
- 全てのメニュー、ラベル、メッセージが日本語で提供されており、直感的に操作できます。
